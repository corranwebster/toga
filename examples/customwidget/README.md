# Document

Test app for adding custom widgets to Toga.

This will display a window with a custom Label-like widget.

## Quickstart

To run this example with the native backend:

```
$ python -m pip install toga
$ python -m pip install -e ./helloworld
$ python -m customwidget
```

To run with the Textual backend
```
$ python -m pip install toga[textual]
$ python -m pip install -e ./helloworld
$ TOGA_BACKEND=toga_textual python -m customwidget
```
