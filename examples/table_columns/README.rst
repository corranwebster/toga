# Table COlumns

Test app for the
[Table widget](https://toga.beeware.org/en/stable/reference/api/widgets/table.html)
showing the use of custom Column objects. The example also shows how you can format numeric and time data in locale-appropriate ways.

## Quickstart

To run this example:

```
$ python -m pip install toga babel
$ python -m table_columns
```
